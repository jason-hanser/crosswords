Crossword Comparison¶

Introduction¶

This was another project I did when I was first starting to play around with python and trying to get better at string and list manipulation. I enjoy doing crosswords and have always been curious if crossword authors ever reuse blocks or patterns from older puzzles. To test that, I built a function to compare crosswords to identify shared substructures - or "motifs" as I call them here.

NOTE: A while back a software engineer was doing similar analysis and uncovered a plagiarism scandal in the crossword world that made national news. By analyzing thousands of crosswords from multiple publishers, they discovered that hundreds of puzzles - all edited by same person - that were either duplicates or near duplicates of older puzzles. When republished, the "new" puzzle was attributed to different author - often fictious. While that analysis was focused on identifying wholesale plagiarism, my approach was to look for repeated motifs.

Data & Background¶

To get started, I loaded a few packages along with the crossword data. The data set for this project was compiled by Dylan O'Shea and hosted on his github. It includes more than 14,000 crossword puzzles published by the New York Times between 1976 and 2018 - with a few gaps in coverage. The crosswords are stored as json files and organized into folders according to when the puzzle was published.

Comparing Crossword Puzzles¶

The function below compares two crossword puzzles by looking for shared motifs. First, the function breaks down each puzzle into a series of motifs, where the size of the motif is determined the the window size argument. The design is effectively a sliding window algorithm. Starting in the top left corner of the crossword, we record the values in that window. We then move the window one column to the right and repeat the process. When we reach the end of the row, we move one column down and repeat the process. Then, the function compares all of the motifs in the first crossword to all of the motifs in the second puzzle. Any repeated motifs are returned along with the author, date, and grid of both puzzles.

def crossword_comparison(crossword_1, crossword_2, window_size = 4):
    
    ## extracting data from the older crossword

    crossword_1_author = crossword_1['author']
    crossword_1_date   = crossword_1['date']

    crossword_1_grid   = crossword_1['grid']
    crossword_1_cols   = crossword_1['size']['cols']
    crossword_1_rows   = crossword_1['size']['rows']

    ## extracting data from the newer crossword

    crossword_2_author = crossword_2['author']
    crossword_2_date   = crossword_2['date']

    crossword_2_grid   = crossword_2['grid']
    crossword_2_cols   = crossword_2['size']['cols']
    crossword_2_rows   = crossword_2['size']['rows']

    ## crossword 1 motifs

    crossword_1_motifs = []

    for i in range(0, (1 + crossword_1_cols - window_size)):
        
        for j in range(0, (1 + crossword_1_rows - window_size)):
                    
            temp_motif = []
                    
            for k in range(0, window_size):
                
                temp_motif_item = crossword_1_grid[(j+(i*crossword_1_rows)+(k*crossword_1_rows)):((j+(i*crossword_1_rows)+(k*crossword_1_rows))+window_size)]
                
                temp_motif = temp_motif + temp_motif_item
                
            crossword_1_motifs.append(temp_motif)


    ## crossword 2 motifs

    crossword_2_motifs = []

    for i in range(0, (1 + crossword_2_cols - window_size)):
        
        for j in range(0, (1 + crossword_2_rows - window_size)):
                    
            temp_motif = []
                    
            for k in range(0, window_size):
                
                temp_motif_item = crossword_2_grid[(j+(i*crossword_2_rows)+(k*crossword_2_rows)):((j+(i*crossword_2_rows)+(k*crossword_2_rows))+window_size)]
                
                temp_motif = temp_motif + temp_motif_item
                
            crossword_2_motifs.append(temp_motif)

   
    ## checking to see if any motiffs in the focal crossword are in other crossword
   
    temp_copied_motifs = [x for x in crossword_2_motifs if x in crossword_1_motifs and set(x) != set('.')]

    if len(temp_copied_motifs) > 0:
        
        if pd.to_datetime(crossword_1_date) < pd.to_datetime(crossword_2_date):
        
            old_crossword_author = crossword_1_author
            old_crossword_date   = crossword_1_date
            old_crossword_grid   = crossword_1_grid
            old_crossword_cols   = crossword_1_cols
            
            new_crossword_author = crossword_2_author
            new_crossword_date   = crossword_2_date
            new_crossword_grid   = crossword_2_grid
            new_crossword_cols   = crossword_2_cols
        
            reused_motifs = [[old_crossword_author, old_crossword_date, old_crossword_grid, old_crossword_cols], [new_crossword_author, new_crossword_date, new_crossword_grid, new_crossword_cols], temp_copied_motifs]
        
            return reused_motifs
        
        else:
            
            old_crossword_author = crossword_2_author
            old_crossword_date   = crossword_2_date
            old_crossword_grid   = crossword_2_grid
            old_crossword_cols   = crossword_2_cols
            
            new_crossword_author = crossword_1_author
            new_crossword_date   = crossword_1_date
            new_crossword_grid   = crossword_1_grid
            new_crossword_cols   = crossword_1_cols
            
            reused_motifs = [[old_crossword_author, old_crossword_date, old_crossword_grid, old_crossword_cols], [new_crossword_author, new_crossword_date, new_crossword_grid, new_crossword_cols], temp_copied_motifs]
            
            return reused_motifs
